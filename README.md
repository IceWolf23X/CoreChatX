# CoreChatX

CoreChatX is a complete communication suite for Minecraft servers.

All official information about the plugin is now available on the CoreChatX website:

> **Website:** https://icewolf23x.github.io/CoreChatX-website/

The website contains the complete and up-to-date information about:

- plugin overview
- features
- installation
- configuration
- FAQ
- troubleshooting
- downloads / releases
- platform support
- Paper setup
- Velocity setup
- Discord and Telegram bridge setup

This GitHub repository is kept only for issue tracking.

---

## Issue Tracking

Use the GitHub **Issues** section to report problems, request improvements, or track known bugs.

Before opening an issue, please check the official website first:

> https://icewolf23x.github.io/CoreChatX-website/

When opening an issue, include as much useful information as possible.

For bug reports, include:

- CoreChatX version
- server platform
- Paper version, if relevant
- Velocity version, if relevant
- Java version
- installed dependencies, such as LuckPerms or PlaceholderAPI
- relevant config sections
- console errors or stack traces
- clear steps to reproduce the issue
- expected behavior
- actual behavior

For feature requests, include:

- what the feature should do
- why it would be useful
- any examples of the desired behavior
